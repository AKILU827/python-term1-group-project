Grade System Program

Project Overview
A simple Python program that accepts 5 marks from users, calculates the average, 
and outputs the corresponding letter grade based on a predefined grading scale.

## Grading Scale
| Score Range | Grade |
|-------------|-------|
| 80+         | A     |
| 70–79       | B     |
| 60–69       | C     |
| 50–59       | D     |
| Below 50    | F     |

Features
- The program accepts 5 numerical marks from user input
- Input validation (ensures marks are between 0-100)
- Calculates the average of the 5 marks
- Determines the appropriate letter grade
- Displays formatted results
